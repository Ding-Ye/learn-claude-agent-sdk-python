# Upstream readings

Annotated excerpts of `claude-agent-sdk-python` pinned to commit [`c352a50`](https://github.com/anthropics/claude-agent-sdk-python/tree/c352a509929a712de65637cbafafcc3a1e3ba4f6).

The actual upstream code is intentionally **not committed** here — see [`docs/zh/appendix-b.md`](../docs/zh/appendix-b.md) or [`docs/en/appendix-b.md`](../docs/en/appendix-b.md) for a map of every concept ↔ upstream file:lines.

To read the upstream alongside your Go:

```bash
git clone --depth 1 \
  https://github.com/anthropics/claude-agent-sdk-python.git \
  .learn/upstream
# now Read .learn/upstream/src/claude_agent_sdk/... as needed
```

The original cloned tree lived in `.learn/upstream/` during this repo's generation; that path is in `.gitignore` so the upstream MIT code isn't redistributed verbatim. Only the excerpts called out in the chapter docs are quoted.
