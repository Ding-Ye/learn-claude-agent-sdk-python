# s04 — Query（一次性）

## Problem

组合问题：怎么把 transport（s02）和 parser（s03）拼成一个函数，告诉它"发这个 prompt、给我 message 流"？

## Solution

`Query(ctx, prompt, opts) (<-chan Message, <-chan error)`。内部：起进程 → 写 JSON envelope → 关 stdin → 抽 stdout → parse → 推到 channel → 见 `ResultMessage` 就停。

## How It Works

我们把 options 当成 JSON envelope 写到第一行 stdin（真 CLI 也是这么收的）。关 stdin 表示"没新轮次了"。一个 goroutine 抽 stdout 喂 channel；一见 `ResultMessage` 就 close channel + `cmd.Wait()`。

## What Changed

- 新模块 `agents/s04-query`，前三章的三块（类型、transport、parser）全部内联。
- 公开面：`Query`、`Options`。
